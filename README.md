# Generic-Data-Structure

This C program implements a **generic list** using `void*` pointers, allowing it to store mixed data types (integers, characters, and student structures) within the same list. It provides core functions to create the list, insert and delete nodes from both ends, and print memory details, all of which are demonstrated sequentially in the `main` function.

```bash
gcc list_of_data -o lod
./lod
```
